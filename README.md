# StockMate Order System

소상공인 매장 발주·재고 관리 서비스 MVP입니다.

## 실행 방법

```bash
npm install
npm run dev
```

기본 주소:

```text
http://localhost:3000
```

3000번 포트가 이미 사용 중이면:

```bash
PORT=3001 npm run dev
```

## 샘플 계정

| 권한 | 계정 | 비밀번호 |
| --- | --- | --- |
| 매장 | 성수 매장 | 1234 |
| 본사 | 본사 관리자 | 1234 |

## 구현 기능

- 로그인 및 권한 구분
- 운영 대시보드
- 발주 신청
- 발주 승인 / 반려
- 재고 수량 / 안전 재고 수정
- 부족 재고 표시
- 매장별 발주 통계
- CSV 다운로드
- 매장 전용 화면 / 관리자 전용 화면 분리
- 발주 수량 `-5`, `-1`, `+1`, `+5` 조절
- 수량 조절 시 카드 애니메이션

## 현재 저장 방식

현재는 `data/db.json` 파일에 데이터를 저장합니다.
실서비스 단계에서는 PostgreSQL 같은 데이터베이스로 교체하는 것이 좋습니다.
